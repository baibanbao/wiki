## GFM!

> github 其实是伪装成 git 仓库的 md 编辑平台

gh 对 md 是狂热的

从发布之初就宣称: [GitHub Flavored Markdown](https://guides.github.com/features/mastering-markdown/)

并将这种狂热传染给了全球开发者, 以致到后来所有 md 编辑器,都得有 github 风格的渲染支持.
这种狂热甚至于高于对 git 的支持,

这也是为什么要将 gh 对 md 的支持, 放在 git 的支持之前来强调
